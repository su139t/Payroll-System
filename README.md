# 💼 Payroll Management System

A comprehensive **Payroll Management System** developed using **ASP.NET MVC** that streamlines employee management, attendance tracking, salary processing, and payroll approvals. The system provides an efficient way for organizations to manage payroll operations while reducing manual effort and improving accuracy.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/Payroll-System.git
```

---

## Open Project

Open the solution in **Visual Studio**.

---

## Configure Database

Update the connection string in:

```
Web.config
```

Example:

```xml
<connectionStrings>
    <add name="PayrollDB"
         connectionString="Your SQL Server Connection String"/>
</connectionStrings>
```

---

## Restore Packages

Restore NuGet packages from Visual Studio.

---

## Run Project

Press

```
F5
```

or

```
Ctrl + F5
```
